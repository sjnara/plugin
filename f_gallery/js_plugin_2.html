제이쿼리 플러그인_팬시박스 갤러리 구현하기

  1. image 폴더에 작은 썸네일 사진(s)과 큰 사진(b)을 삽입하여 저장한다.
  2. html태그요소 a에 작은 사진을 클릭시 큰사진이 나오도록 하아퍼 링크한다.
  3. 제이쿼리 라이브러리 연결(1.8버전)
  4. 팬시박스 라이브러리 js연결
  5. 팬시박스 css 연결
  6. 팬시박스 관련 이미지 폴더
  7. 팬시박스 효과 스크립트 소스삽입(example1,example2~8)
  8. a태그에 효과종류 id명을 작성하고 rel='example_group'을 삽입하여 하나의 그룹으로 묶어줌
